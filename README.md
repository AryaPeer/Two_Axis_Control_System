# Two Axis Motor Control Project

A two-axis motor control system built on the STM32 NUCLEO-F401RE with the X-NUCLEO-IHM02A1 stepper motor driver shield. The system drives two stepper motors to move a platform to a target location, with limit switch safeties to prevent over-travel in either axis. A potentiometer controls motor direction and a button toggles which motor is active.
